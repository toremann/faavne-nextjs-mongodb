## Frontend for Faavne

Hosted at [www.faavne.no](https://www.faavne.no)

Faavne finds dividend stocks and rates them.

Frontend deployed with Vercel

- Typescript
- Nextjs
- Bootstrap
- Bootstrap icons
- Chartjs
- React Bootstrap
- Popper

Backend deployed with Railway

- Nodejs
- MongoDB
- Puppeteer
- Node-schedule
